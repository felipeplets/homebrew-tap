# Homebrew Tap for Rooms

This is the official Homebrew tap for [rooms](https://github.com/felipeplets/rooms), a terminal UI to manage Git worktrees.

## Installation

```bash
brew tap felipeplets/tap
brew install rooms
```

Or install directly:

```bash
brew install felipeplets/tap/rooms
```

## Updating

```bash
brew update
brew upgrade rooms
```

## Uninstalling

```bash
brew uninstall rooms
brew untap felipeplets/tap  # optional
```

## About Rooms

Rooms is a keyboard-driven terminal UI for managing Git worktrees. It provides:

- Two-panel interface: sidebar list of rooms + embedded terminal
- Interactive and silent room creation
- Safe room removal with confirmations
- Post-create command support

For more information, see the [rooms repository](https://github.com/felipeplets/rooms).
